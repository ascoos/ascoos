# Οικοσύστημα Ascoos

Το Ascoos αποτελεί ένα ολοκληρωμένο οικοσύστημα λογισμικού που περιλαμβάνει τεχνολογίες, εργαλεία και εφαρμογές για Web, IoT και AI περιβάλλοντα.  
Στόχος του είναι η παροχή σύγχρονων, αποδοτικών και επεκτάσιμων λύσεων για προγραμματιστές και οργανισμούς που αναζητούν υψηλή απόδοση, modular αρχιτεκτονική και τεχνολογική καινοτομία.

Το παρόν GitHub account φιλοξενεί έργα σε εξέλιξη, βιβλιοθήκες ανοικτού κώδικα και υποστηρικτικά εργαλεία που αποτελούν μέρος του ευρύτερου οικοσυστήματος.

---

## 🔹 Ιστορικό & Εξέλιξη

Η ανάπτυξη του οικοσυστήματος Ascoos ξεκίνησε το 2007–2010 με το **Ascoos CMS 10**, το πρώτο παγκοσμίως Windows‑like (Desktop‑like) CMS.  
Το Ascoos CMS 10 αξιοποιούσε εκτεταμένα JSON και jQuery σε μια εποχή όπου κυριαρχούσαν XML‑based τεχνολογίες και prototype.js, εισάγοντας μια πρωτοποριακή desktop‑like εμπειρία διαχείρισης.

Η αρχιτεκτονική και οι τεχνολογικές επιλογές του Ascoos CMS 10 αποτέλεσαν τη βάση για τη φιλοσοφία του σημερινού **Ascoos OS**:  
χαμηλού επιπέδου σχεδιασμός, modular υποδομή, desktop‑like διεπαφές και δυνατότητα λειτουργίας ως θεμέλιο για frameworks, CMS, APIs και σύνθετες εφαρμογές.

## 🔹 Desktop Εφαρμογές

### **Ascoos Web Extended Studio (AWES)**  
Ένα ολοκληρωμένο, φορητό IDE για ανάπτυξη web εφαρμογών, το οποίο περιλαμβάνει:

- πολλαπλούς Web, DB και FTP servers,  
- ενσωματωμένα εργαλεία (editors, browsers κ.ά.),  
- υποστήριξη πολλαπλών εκδόσεων PHP,  
- οπτικά εργαλεία διαχείρισης και debugging,  
- πλήρως portable λειτουργία.

Το AWES λειτουργεί ως πλήρης σουίτα ανάπτυξης και όχι απλώς ως server runner.

## 🔹 Κύρια Έργα

| Έργο | Περιγραφή |
|------|-----------|
| [**Ascoos OS**](https://github.com/ascoos/os) | Χαμηλού επιπέδου **Web5/WebAI Kernel** σε pure‑PHP, με ενσωματωμένη AI, Macro Engine (DSL/AST) και τεχνολογίες όπως `CiC`, `OTA`, `DUAL`, `LibIn`. Αποτελεί θεμέλιο για frameworks, CMS, APIs, IoT και σύνθετες εφαρμογές. |
| [**Ascoos Web Extended Studio (AWES)**](https://github.com/ascoos/awes) | Φορητό IDE για ανάπτυξη και έλεγχο web εφαρμογών σε πολλαπλές εκδόσεις PHP και βάσεων δεδομένων, με οπτικά εργαλεία και επεκτασιμότητα. |
| [**JML (JSON-style Markup Language)**](https://github.com/ascoos/jml/blob/main/README-GR.md) | Ελαφρύ markup format εμπνευσμένο από JSON και DSLs, σχεδιασμένο για γρήγορη δημιουργία UI templates στο Ascoos OS, με AST υποστήριξη και πλήρη UTF‑8 συμβατότητα. |
| [**BootLib**](https://github.com/ascoos/bootlib) | UI Framework με εκτενή βιβλιοθήκη διεπαφής, υποστηριζόμενο από το JavaScript Framework `JAscoos`. |
| [**JSQLDB**](https://github.com/ascoos/jsqldb) | Υβριδική JSON‑SQL μηχανή βάσης δεδομένων με έμφαση στην ταχύτητα, modularity και PHP‑native queries. |
| [**Greek Spellchecker**](https://addons.mozilla.org/el/firefox/addon/greek-spellchecker/) | Πρόσθετο για τον Firefox για ορθογραφικό έλεγχο στην ελληνική γλώσσα. |
| [**phpBCL**](https://github.com/ascoos/phpbcl) | Βιβλιοθήκη συμβατότητας PHP για παλαιότερες εκδόσεις. |
| [**WIC (Web Images Cryptography)**](https://github.com/ascoos/wic) | Νέα μορφή αρχείου εικόνας (`*.wic`) με υψηλή συμπίεση και αυξημένη ασφάλεια για χρήση στο web. |
| [**WebAI**](https://github.com/ascoos/WebAI/blob/main/README-EL.md) | Το WebAI αποτελεί αρχιτεκτονική όπου το ίδιο το διαδίκτυο λειτουργεί ως νοήμων οντότητα, και όχι απλώς ως σύνολο AI προσθηκών. |

## 🔹 Τεχνολογική Υπεροχή Ascoos OS

Το **Ascoos OS** δεν είναι framework· είναι **χαμηλού επιπέδου Kernel** που λειτουργεί κάτω από το επίπεδο των σύγχρονων PHP frameworks.  
Ο διαθέσιμος συγκριτικός πίνακας παρουσιάζει τις διαφορές ανάμεσα στο Ascoos OS και τα frameworks, αναδεικνύοντας τον ρόλο του ως θεμέλιο πάνω στο οποίο μπορούν να χτιστούν frameworks, CMS και σύνθετες εφαρμογές.

Δείτε τον [συγκριτικό πίνακα](https://github.com/ascoos/os/blob/main/COMPARE-GR.md) για περισσότερες λεπτομέρειες.
